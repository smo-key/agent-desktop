Agent Desktop

Manage AI coding agents — for people who run lots of agents but only want to think about one thing at a time.

Install on macOS or Linux

curl -fsSL https://smo-key.github.io/agent-desktop/install.sh | sh

Install on Windows

irm https://smo-key.github.io/agent-desktop/install.ps1 | iex

Prefer to download by hand, or on an Intel Mac? Grab an installer from the latest release. The scripts auto-detect macOS (Apple Silicon), Linux (x86_64 / arm64), and Windows (x64); Intel-Mac builds are coming soon. Both verify the download's checksum before installing.
