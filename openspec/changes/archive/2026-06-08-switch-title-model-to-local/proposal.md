## Why

Each agent card in the overview shows a short FOCUS title ("SKIPA-45: Fix
Feature", "Improve frontend dialog handling") derived from the user's messages.
That title was generated by shelling out to `claude -p --model haiku` — a network
round-trip to a hosted model on every change of the user's messages. The app
already runs an on-device LLM sidecar (`llama-server` loading the Qwen3 polish
model) for transcript polish; routing titles through the same local model removes
the per-title network dependency and hosted-model cost, and keeps the user's
session content on-device.

## What Changes

- The session-title generation behind the `session_focus` command now calls the
  **local model** (the `llama-server` sidecar loading the
  [`crate::models::POLISH`] Qwen3 model) instead of `claude -p --model haiku`.
  The sidecar lazy-starts on first request and is shared with transcript polish.
- The prompt is bounded to fit the sidecar's modest context (last 20 user
  messages, each clipped) and the title carries the same constraints as before
  (≤6 words, only the title, no quotes/punctuation/preamble).
- Any Qwen3 `<think>…</think>` reasoning that leaks into the response is stripped
  before the title is returned; reasoning is also disabled in the request.
- Graceful degradation is unchanged: when the local model is absent or the call
  fails, the command returns an error and the overview keeps the previous title.
  When the user has sent no messages, no title is produced.

## Capabilities

### New Capabilities
- `session-titles`: the overview's per-agent focus title, generated on-device from
  the user's transcript messages and gated on a user-messages hash.

## Impact

- **Backend** — `src-tauri/src/polish.rs`: extract a shared `chat_complete` core
  from `voice_polish`; add `build_title_body` + `TITLE_SYSTEM_PROMPT`.
- **Backend** — `src-tauri/src/lib.rs`: `session_focus` becomes an async command
  that runs the title completion through the local model; add the pure
  `clean_title` / `strip_think_blocks` post-processing helpers.
- **Frontend** — comment-only updates in the overview titles store and callers
  (no behavior change; the TS contract for `session_focus` is unchanged).
- No new dependency, model download, or protocol change — the title reuses the
  polish model already provisioned for transcript polish.
